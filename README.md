# Installation

Build dependencies:
- [Setuptools](https://setuptools.readthedocs.io/en/latest/)
- [Bash](https://www.gnu.org/software/bash/)

Run `make`, `make install`.  The usual `DESTDIR`, etc. arguments to `make` are
supported.

# Dependencies

- Python 3 (>= 3.7)
